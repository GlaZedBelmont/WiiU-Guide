Homebrew Launcher

Please check that your console is in version 5.5.3 or 5.5.4.

Before you can hack your WiiU you need to download all its files and put them on your SD card.

Homebrew Launcher

Homebrew launcher is the most important application, it allows you to launch all the others. You can download it by clicking here.

Once the archive is downloaded, simply extract the WiiU Folder from the root of the SD Card.

Payload.elf

Payload.elf is a micro software that allows you to launch the homebrew launcher with the new exploits. You can download it by clicking here.

Once you’ve downloaded the archive, just put it in the wiiu folder at the root of your sd card.

If the file is misplaced, or is non-existent, the exploit will not work.

Homebrew App Store

In order to be able to test if any function correctly we need an application.

Is there an app more essential than Homebrew App Store? This application allows you to download Homebrew apps directly from your wiiu, without having to take out your SD card.

To download it, please click here.

Normally, in the archive you will need to find a wiiu folder, a json file, a manifest file and two images.

To add the application, simply move the wiiu folder from the archive to the root of the SD card. If you already have the wiiu folder, it should merge.

Test

Now it’s time to see if all your efforts are working.

  1. If your wiiu is on, turn it off.
  2. Eject your SD card, then insert it into your wiiu.
  3. Turn on your console
  4. Launch the console’s web browser.
  5. Go to u.wiidb.de.
  6. Click on HAXX
  7. One page should load. If you look at the bottom left, you should see your mii swing. If it stops moving, an error has occurred. You should then turn the console off and back on again. If your mii stops and a wiiu logo launches in the lower right corner, you’ve done it.

Error handling

If you have a black screen with text written on it, this section is for you. Otherwise you can go to the next step.

If you see one of these screens it means that the payload.elf has been loaded.

  • FSGetMountSource Failed: This means that your SD card, is not in fat32, it has to be reformatted.
  • FSOpenFile Failed: This means he didn’t find the payload.elf file, just check if it’s in the right place or if it exists.
  • The browser crashed: As mentioned above, this happens, just delete the browser options and restart the exploit.

Finish

Here you have installed the strict minimum for wiiu hacks.
You can now follow the other tutorials.
